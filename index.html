Current filtering rules:
- Filter out: Map, Save Game, Map Theme, District Style, Scenario, Intersection
- Filter out: assets with only one image. Authors with are allowed
- Map Themes from authors with are allowed
- This is unavoidably subjective but more authors may be added to the list

Report issues(bugs, suggestion, questions, etc.) on Github: https://github.com/sway2020/sway2020.github.io
